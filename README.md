# PulseKit ⚡

> Production-ready Python scripts, API wrappers, and automation tools for developers and operators who need to move fast. No enterprise bloat — just sharp, deployable code that gets the job done.

This repository contains the official landing page and script directory showcase for **PulseKit**, built as a responsive static website featuring a sleek, glowing dark aesthetic, live code snippets, and interactive copy utilities.

## 🚀 Features

- **0% Enterprise Bloat**: Pure, lightweight code blocks with zero unnecessary dependencies.
- **Copy-and-Deploy Playground**: An interactive code gallery showcasing resilient scrapers, webhook notification servers, Discord alert scripts, and background schedulers.
- **Highly Responsive**: Fully optimized for mobile, tablet, and desktop viewing.
- **Seamless Integration**: Styled directly with vanilla CSS using modern design elements (glassmorphism, radial ambient lighting, fluid micro-interactions).

## 📂 Repository Contents

- `index.html` - Semantic structure including CDN resources (Google Fonts, Lucide Icons, PrismJS).
- `style.css` - Custom styling tokens, layouts, visual effects, and fluid animations.
- `app.js` - Interactive front-end functionality (tab controller, clipboard manager, mobile side navigation, dynamic scroll transitions).
- `.gitignore` - Standard configuration to keep the repository clean.
